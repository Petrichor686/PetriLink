# PetriLink

PetriLink 是一款面向嵌入式开发的串口、TCP/UDP 网络调试与固件下载工具。

它适合开发板、最小系统板、传感器和网络设备的日常调试，采用便携式发布方式，无需安装 Qt，解压后即可运行。

## 主要功能

- 串口收发与实时数据显示
- TCP 客户端、TCP 服务端和 UDP 调试
- 文本、ASCII、HEX 多种数据显示模式
- RX/TX 时间戳和终端风格日志
- 单条发送、多条发送和定时发送
- 双通道并行调试
- STM32/GD32 兼容 USART Bootloader 固件下载
- BIN/HEX 固件文件、Flash 擦除、写入和下载后校验

## 使用方法

1. 双击 `PetriLink.exe`。
2. 在左侧选择连接类型和参数。
3. 串口调试时选择 COM 端口和波特率，点击“打开串口”。
4. 在接收区选择文本、ASCII 或 HEX 显示模式。
5. 在发送区输入数据，或使用 HEX 发送、定时发送功能。

## 固件下载

使用“固件下载”前，请先让 STM32/GD32 芯片进入系统 Bootloader：

1. `BOOT0` 置高。
2. 复位芯片。
3. 在 PetriLink 中选择对应串口。
4. 点击“固件下载”，选择 `.bin` 或 `.hex` 文件。
5. 下载完成后将 `BOOT0` 置低，再次复位运行程序。

普通 STM32 工程的 BIN 起始地址通常是：

```text
0x08000000
```

HEX 文件通常已经包含地址信息。若工程使用了自定义 Bootloader，请以工程链接脚本中的应用地址为准。

## 运行要求

- Windows 10/11 64 位
- 不需要安装 Qt 或 FlyMcu
- 使用 CH340、CP2102 等 USB 转串口模块时，电脑可能需要安装对应的 USB 串口驱动

## 目录说明

请保持 `PetriLink.exe` 与 Qt DLL、`platforms`、`imageformats` 等子目录一起使用，不要只单独复制 `PetriLink.exe`。
